RANDY

Solution to a programming problem posed by PROTEUS Technologies.

RANDY demonstrates random number generation. It comes with a GUI, but is better-looking on the inside.

To run:

    java -jar randy.jar

To run, specifying alternate random number generators:

    java -jar randy.jar net.ech.randy.KISSRandomCore

    java -jar randy.jar net.ech.randy.LinearCongruentialRandomCore

To run, specifying a user-defined random number generator:

    java -cp 'randy.jar;{myclasses}' net.ech.randy.Main {myRNG}

Where:

  • {myRNG} is the qualified classname of the random number generator,
  • {myclasses} is the CLASSPATH component where {myRNG} is found.

A user-defined random number generator must implement the net.ech.randy.IRandomCore interface.
